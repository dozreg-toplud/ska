## Hoon
This repository contains code written in Hoon. Hoon is a functional programming language which compiles to Nock.

The documentation of Hoon's syntax and semantics are at https://docs.urbit.org, along with the documentation of Nock.

Command line compilation without an Urbit ship can be accomplished using the `hoonc` crate from the [Nockchain repo](https://github.com/nockchain/nockchain).

Read the docs and the existing Hoon in this repo. Be careful not to hallucinate syntax, as Hoon syntax is very unconventional.
